### Python

- 가변형(Mutable) vs. 불변형(Immutable)

  - 가변형은 인덱싱을 통해서 부분적인 변경이 가능

- 반복 가능(Iterable)

- 시퀀스(Sequence) vs. 논 시퀀스(Non-Sequence)

  - 시퀀스는 인덱싱이 가능함(Subscriptable)

  

- Booltype 연산

  - `True+정수` = `정수+1`

  - `False+정수` = `정수`



- `int(n/2) vs. //`(음수일 경우에 조심)
  - 둘의 계산값은 음수일 경우에 다르므로 유의해야함

```python
# 예시
print(int(-5/2)) # -2
print(-5//2) # -3
```
